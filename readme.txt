CopyCraft is an AI-powered WordPress plugin that helps you create accurate and compelling product descriptions for your WooCommerce products.

== Description ==

CopyCraft is designed to make the task of writing product descriptions easier and more efficient. It uses advanced AI algorithms to analyze your product and generate descriptions that are accurate, informative, and engaging. With CopyCraft, you can save time and effort while still ensuring that your product descriptions are of the highest quality.

Uses OpenAI's GPT-3 API to generate product descriptions.

Features:

- AI-powered product description generator
- Accurate and compelling descriptions
- Easy to use interface
- Works with WooCommerce

== Installation ==

- Install the plugin from the WordPress plugin repository or by uploading the .zip file to your site.
- Activate the plugin.
- Go to the plugin settings page and enter your OpenAI API key.
- Use the plugin to generate descriptions for your WooCommerce products.

== Frequently Asked Questions ==

Q: Do I need an API key to use the plugin?
A: Yes, you will need to obtain an OpenAI API key in order to use the plugin. You can do this by signing up for a free trial on our website.

Q: Is the plugin compatible with other e-commerce platforms besides WooCommerce?
A: Currently, the plugin is only compatible with WooCommerce. However, we are working on adding support for other platforms in the future.

Q: Can I customize the generated product descriptions?
A: Yes, you can edit the generated descriptions to suit your needs. The plugin provides a convenient WYSIWYG editor for this purpose.
